bipandoc

A synchronizer between document formats. It synchronizes content-relevent information from the source document into the target document while trying to retain the stylish information in the synchronization target as much as possible. When the synchronization target is empty, it can be treated as a conventional document converter like the well known pandoc. Since it is implemented with the BiGUL library, it guarantees that if you synchronize the target back into the source, the result will be identical with the original source, i.e. `put source (put target source) == source`.
